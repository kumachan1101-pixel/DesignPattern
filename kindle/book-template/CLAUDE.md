# CLAUDE.md — Kindle本テンプレート（汎用）
# 新しいKindle本を作る際はこのテンプレートをコピーして使う

## このテンプレートの使い方

1. このフォルダ（book-template）をコピーして新しい本のフォルダを作る
2. 新しい本の CLAUDE.md の「このプロジェクトが作るもの」を書き換える
3. templates/ に本の構造に合ったテンプレートを作る
4. agents/ のエージェントを本の構造に合わせて調整する

## Kindle本共通のチェック項目

### Kindle制限
- コードブロック：1行80文字以内
- 脚注：使わない（Kindleでの表示が不安定）
- 表：列数は最大4列まで
- Mermaid図：テキストでも意味が伝わる説明を必ず添える

### 読みやすさ
- 1段落：5〜7行以内
- コードブロックの前後：必ず説明文を置く
- 箇条書きが5項目以上続く場合は散文への変換を検討する

### Markdown品質
- 太字・コードブロックの閉じ忘れがないか
- 執筆指示コメント（`<!-- -->`）が出力に残っていないか
- TODO・メモ書きが残っていないか

### 著者の人格
- 「〜すべき」「〜べきです」がないか
- 否定的なラベルがないか
- 著者の口癖が自然に散りばめられているか

## ファイル構成（このテンプレートの構造）

```
book-template/
├── CLAUDE.md              ← このファイル（コピーして使う）
├── .claude/hooks/
│   └── post-chapter.sh    ← 章生成後の品質チェック
├── agents/
│   ├── chapter-agent.md   ← 章生成担当
│   └── review-agent.md    ← レビュー担当
├── skills/
│   └── chapter-writer.md  ← 章執筆手順（本の構造に合わせて書き換える）
├── templates/
│   └── chapter-template.md ← 章テンプレート（本の構造に合わせて書き換える）
└── output/
```
