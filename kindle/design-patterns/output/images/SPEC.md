# SVG 画像仕様書

## cable-matrix.svg ― 2×2 接続マトリクス全体図

### 目的
chapter00_1.md の INFO ブロック直下に置く。
「具体/抽象 × 直接/間接」4種類の接続をブロック形式の機材図で一覧する。

### キャンバス
- サイズ: `width="680" height="460"`
- 背景: `fill="#f8f9fa"` round `rx="6"` border `#d0d0d0`

### グリッドレイアウト

```
[0,0]    x=0..89   (ラベル列)
[0,1]    x=90..384 (左セル列)
[0,2]    x=385..679(右セル列)

y=0..49  ヘッダー行
y=50..254 上セル行（具体）
y=255..459 下セル行（抽象）
```

分割線:
- 垂直: `x=385` stroke `#bbb` width=2
- 水平: `y=255` stroke `#bbb` width=2
- 外枠: `x=90 y=50 width=590 height=410` stroke `#bbb`

### デバイスブロック規格

**iPhone** (`rx=8`, 塗り `#ececec`, 線 `#999` 1.5px)
- 本体: `52×90`
- 画面: 本体内 `margin 6px` 塗り `#aab8c2`
- ホームバー: 下端 `16×4 rx=2 #90a4ae`
- Lightning ポート（右端）: `5×7 rx=1 #888`

**MacBook** (`rx=4`, 塗り `#d5d5d5`, 線 `#999` 1.5px)
- 本体: `90×62`
- 画面: 本体内 `margin 4px` 塗り `#aab8c2`
- ベース: 本体幅 +16px, 高さ `9`, `rx=3 #c0c0c0`

**Hub** (`rx=6`, 塗り `#e8f5e9`, 線 `#43a047` 2.5px)
- 本体: `56×62`
- ポート: 右端に `6×5 rx=2 #43a047` を3個縦並べ（間隔13px）

**変換アダプター** (`rx=5`, 塗り `#fff8e1`, 線 `#ff8f00` 2px)
- 本体: `48×28`
- テキスト: 「変換」`8px #bf6000 bold` / 「Adapter」`7px #bf6000`

**USB-C 汎用デバイス** (`rx=8`, 塗り `#e3f2fd`, 線 `#64b5f6` 2px)
- 本体: `55×82`
- 画面: 内 `margin 6px` 塗り `#82b1d4`

### コネクタ・ケーブル規格

**Lightning コネクタ**: `11×9 rx=1.5` 塗り `#ff8f00` 線 `#bf6000 0.8px`

**USB-C コネクタ**: `10×9 rx=4` 塗り `#1976d2` 線 `#0d47a1 0.8px`
（楕円に見える程度の `rx` が重要）

**ケーブル線**: `stroke-width=5`（Lightning 橙・USB-C 青）

### タイポグラフィ
- フォント: `'Hiragino Sans', 'Noto Sans JP', sans-serif`
- ヘッダー軸ラベル: `13px bold #2c3e50`
- セル内デバイスラベル: `9px #555`
- セル内ケーブルラベル: `9px italic` ケーブル色
- セル下部説明テキスト: `10px #777`

### 各セルの構成（テンプレート）

```
y+15  セルタイトル（10px #777 italic）
y+40  ─── 機材ブロック群 ─── （幅の中央揃え）
        [デバイスA] ─コネクタ─ケーブル─コネクタ─ [（アダプター─）デバイスB]
y-20  説明テキスト（10px #777）
```
