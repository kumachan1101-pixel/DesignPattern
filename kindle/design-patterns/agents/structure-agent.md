# agents/structure-agent.md
# 第0章と全体構成を担当するエージェント

---

## 役割

第0章と全体の目次・構成を担当する。
各章エージェントとは独立して動く。

---

## 担当する作業

### 作業1：第0章の生成

1. `../../shared/skills/author-voice.md` を読む
2. `templates/chapter0-template.md` を読む
3. `patterns/*.yaml` を一覧して全パターンの表を生成する
4. 以下の内容を含む第0章を生成する：
   - なぜパターンを覚えても使えないのか（著者の共感から入る）
   - 5ステップの思考プロセスの説明
   - 各章で「変化の種類」だけが変わるという構造の説明
   - 全パターンの一覧表（変化の種類・辿り着くラベル）
   - この本の読み方（どの章から読んでもよい）
5. `output/chapter00.md` に書き出す

### 作業2：目次ファイルの生成

`patterns/*.yaml` から全パターンを読み込み、
`output/toc.md` を生成する。

目次の形式：
```
# 目次

## 第0章　この本の読み方
── デザインパターンは「結果」に過ぎない

## 第1章　Strategyパターン
── 今回の変化：「実行する振る舞いが変わる」

## 第2章　Observerパターン
── 今回の変化：「通知を受け取る相手が変わる」
...
```

---

## 出力

```json
{
  "agent": "structure-agent",
  "status": "complete",
  "outputs": [
    "output/chapter00.md",
    "output/toc.md"
  ]
}
```
