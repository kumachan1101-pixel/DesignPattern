# CLAUDE.md — Kindle × Note 執筆プロジェクト 全体ルール

## 操作範囲（絶対制約）

操作してよいのは `files/` フォルダ配下のみ。
`C:\Users\kumac\OneDrive\デスクトップ\Claude\files\`

- `files/` 外・システムファイル・Claude設定には一切触れない
- 明示的な指示がない限り、既存ファイルを削除しない

以下は実行前にユーザーへ確認する：
- 既存ファイルの大幅な書き換え（構造を変える場合）
- ファイルの削除・新しいフォルダの作成（構造変更を伴う場合）

## 作業開始前の必須確認（この順で読む）

1. `shared/skills/author-voice.md` — 語り口・禁止表現（最重要）
2. 対象プロジェクトの `CLAUDE.md`
3. 使用するスキルファイル（skills/*.md）
4. 読者像：`shared/personas/reader-profiles.md`

## 出力ルール

- `output/` への書き出し以外のファイル変更は明示的な指示がある場合のみ
- 完了報告は必ずJSON形式で出力する
- 執筆指示コメント（`<!-- -->`）を出力ファイルに残さない

## 作業体制（AI マルチエージェント）

このプロジェクトは **Agent ツールを使ったマルチエージェント体制** で作業する。

- 複数章への一括変更・長大なコード書き換えなど、1コンテキストでは重い作業は Agent を起動して並列処理する
- メインスレッドはディレクション・確認・コミットを担い、重い書き込みはサブエージェントに委ねる
- サブエージェントへの指示には「変えてよい箇所」と「変えてはいけない箇所（日本語散文・核心・表構造）」を明示する
- 作業完了後はメインスレッドが差分を確認してからコミット・プッシュする
